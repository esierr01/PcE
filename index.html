REPOSITORIOS GIT / GITHUB

Inicializar GIT en Local para nuestro proyecto

- Primero nos ubicamos en la terminal gitbash en el directorio de nuestro proyecto y escribimos:

git init >> Comando para iniciar Git Local en nuestro proyecto
git add . >> Comando para agregar todos los archivos del proyecto al Git Local
git commit -m "mi primer commit" >> Comando para hacer commit, se coloca comentario
git branch -M main >> Comando para definir la rama del Git, se inicia como main

Inicializar GIT ahora en repositorio en GITHUB (nube)

- Creamos en www.github.com una cuenta y luego el repositorio nuevo donde guardaremos nuestro proyecto y nos ubicamos en la terminal gitbash en el directorio de nuestro proyecto y escribimos:

git remote add origin https://github.com/user/repo.git >> Comando para indicar repositorio (GITHUB) creado
git push -u origin main >> Comando para cargar los archivos del repositorio local al repositorio en GITHUB

Actualizar repositorio en GITHUB (nube)

- Primero nos ubicamos en la terminal gitbash en el directorio de nuestro proyecto y escribimos:

git status >> Comando para ver los commits pendientes
git add . >> Comando para agregar todos los archivos del proyecto nuevos y/o modificados al Git Local
git commit -m "msg descriptivo del commit" >> Comando para hacer commit
git push -u origin main >> Comando para cargar cambios al repositorio en GITHUB
